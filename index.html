🎨 MSE フォント変更

・概要
著名オリカ作成アプリ Magic Set Editor の設定におけるフォントの変更手順について記す.
以下 %MSE% は MSE をインストールしたパスである.

・フォントの追加
所望の OS に MSE で使用したいフォントを追加する. 既にインストール済みであればﾖｼ!
MSE 用フォントとしてインストールするという場合は, %MSE% 下の下記ディレクトリに追加する.
追加先パス：%MSE%\"Magic - Fonts"
例として, MinionPro-Bold を以下で用いるためにインストールしたものとする.

・設定ファイル編集
例として, 旧枠スタイル (magic-old.mse-style) で用いられるフォントをロシア語フォントへ変更する.
なお, 実際に刷られたロシア語版 Retro Frame カードの公式フォントは入手不可である (2026/04 現在).
旧枠スタイル用の設定ファイル (style) を適当なテキストエディタで編集する.
編集前のファイルは適宜 style.bakYYYYMMDD 等で退避しておく.

%MSE%\data\magic-old.mse-style/style:239

  -      name: MagicMedieval
  -      size: 15.5
  +      name: MinionPro-Bold
  +      size: 13
%MSE%\data\magic-old.mse-style/style:284

  -      name: MPlantin
  +      name: MinionPro-Bold
%MSE%\data\magic-old.mse-style/style:335

  -      name: MPlantin
  -      scale down to: 6
  +      name: Times New Roman
  +      scale down to: 10.1

・アプリケーション再起動
設定ファイルの編集が完了したら, ファイルを保存し, mse.exe を再起動する.
見栄えを確認し, フォントやフォントサイズを適宜調整する.

・設定戻し
所望のオリカ作成が完了したら, 画像をエクスポートする. ﾖｼ!
設定ファイル (style) の編集箇所を元に戻す (退避した style.bakYYYYMMDD からリネームする等).

ぐぬぬ... MSE で露語フォント対応をしてみた結果. フォント設定はうまくできたのだけど, やはり代替フォントでは満足ならぬ. 特にテキスト部分は良いのがなくて Times New Roman にしたけど公式フォントに大きく劣る. そのうち AI でフォントも生成できるようになると期待..🐍 pic.twitter.com/0y3tZjT2HW

🧪 Scryfall カラム変更

・概要
著名カード検索アプリ Scryfall の検索結果で表示されるカラム数を変更する手順について記す.

・一行あたりに表示されるカードの枚数を変更する
[要素を検証] (F12) > [Sources] > Ctrl + F > "card-grid-item" より以下の項目を変更. 適宜調整.

scryfall-3e21fe7179294691877b9883246cd6d8b847812cdbbcced28f093c1009390be0.css

      @media screen and (min-width: 920px) {
        .card-grid-item {
  -       width:24.5%
  +       width:6.5%
        }
      }

・カード枚数アイコンのサイズを変更する
この変更は Scryfall のカード検索結果ではなく, Your Decks の Visual View で実施することを想定している.
上記の変更でカード画像サイズが小さくなると, カード枚数アイコンがカード画像サイズに合わなくなる場合がある.
[要素を検証] (F12) > [Sources] > Ctrl + F > "card-grid-item-count" より以下の項目を変更. 適宜調整.

scryfall-3e21fe7179294691877b9883246cd6d8b847812cdbbcced28f093c1009390be0.css

      .card-grid-item-count {
        position: absolute;
  ...
        z-index: 5000;
  -     top: 13.5%;
  -     left: 11%;
  -     padding: 0 5px 0 7px;
  -     height: 22px;
  +     top: 6.5%;
  +     left: 10%;
  +     padding: 0 3px 0 7px;
  +     height: 13px;

Scryfall 画像カードリストのスタイル変更方法 🥸
デフォルトで card-grid-item { width:24.5% } の箇所を減らすと一行に表示されるカードの枚数が増える 🐡 表示されるカードの枚数が増えて左上の枚数表示のアイコン (card-grid-item-count) が相対的に大きくなりすぎる場合はそれも変更する 🐍 pic.twitter.com/Kvedjhj9L3
